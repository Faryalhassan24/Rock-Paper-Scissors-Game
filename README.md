# Stone, Paper, Scissors Game 

A simple and interactive Stone Paper Scissors game built using HTML, CSS, and JavaScript. Play against the computer, track scores, and enjoy smooth animations with a fully responsive UI.

## Features 
1.	Interactive User vs Computer gameplay
2.	Random computer-generated choices
3.	Real-time score updates
4.	Smooth hover and click animations
5.	Responsive layout for all devices
6.	Clean and minimal UI
7.	Beginner-friendly JavaScript logic

 ## Technologies Used 

•	HTML5
•	CSS3
•	JavaScript
